## 👥 Шаблони для ролей (Issues Templates)
Використовуйте ці шаблони для створення тікетів у вашому GitHub Project:
- **[SME Template](sme_template.md)** — для доменних правил (`DR`), бізнес-процесів (`BAU`) та граничних випадків (`EC`).
- **[UX Template](ux_template.md)** — для проектування шляхів користувача (`UF`) та макетів екранів (`WF`).
- **[DEV Template](dev_template.md)** — для технічної деталізації та розбиття Stories на таски (`TT`).
- **[QA Template](qa_template.md)** — для опису атомарних тест-кейсаів (`TC`) та стратегії якості.
- **[OPS Template](ops_template.md)** — для інфраструктурних завдань (`OPS`) та налаштування CI/CD.
