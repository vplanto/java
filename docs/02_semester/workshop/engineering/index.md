# Engineering Workshop Track

Цей розділ містить матеріали для технічних практикумів зі створення Production Ready застосунків на Spring Boot.

## Основний трек (Spring Boot Ecosystem)
*Від фізики Web-процесів до автоматичного деплою в Production.*

- **[P02: Архітектура сучасних Web-застосунків](p02_spring_web_arch.md)** — SSR vs SPA, IoC Container, Spring Boot Philosophy
- **[P03: Zero to Hero — перший Spring Boot сервіс](p03_spring_zero_to_hero.md)** — Spring Initializr, Embedded Tomcat, REST Controller, DTO
- **[P04: Архітектурна гігієна та DI](p04_spring_architecture_di.md)** — Service Layer, Constructor Injection, Separation of Concerns
- **[P05: Production Ready](p05_spring_production_ready.md)** — External Config, Global Exception Handling, HTTP Status Codes
- **[P06: Docker на практиці](p06_docker_practice.md)** — Dockerfile, Docker Compose, Debugging
- **[P07: Hello Cloud — деплой у хмару](p07_spring_cloud_deployment.md)** — PaaS (Render.com), CI/CD, Cold Start, Environment Variables
- **[P08: API Design на практиці](p08_api_practice.md)** — URL-структура, HTTP-статуси, OpenAPI YAML, Breaking Changes
- **[P09: Testing на практиці](p09_testing_practice.md)** — Unit (Mockito), MockMvc, Coverage

## Фінальний проєкт
- **[Посібник з виконання фінальної роботи курсу](p01_final_project_guide.md)** — детальна інструкція, крок за кроком
- **[Vibe Coding Protocol & Оцінювання](vibe_coding.md)** — критерії якості коду, використання AI та архітектурних рішень
