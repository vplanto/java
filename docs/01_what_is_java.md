1.  **–ó—É–ø–∏–Ω–∫–∞ —Å–≤—ñ—Ç—É ("Stop the world"):** –†–æ–±–æ—Ç–∞ –æ—Å–Ω–æ–≤–Ω–æ—ó –ø—Ä–æ–≥—Ä–∞–º–∏ –ø—Ä–∏–∑—É–ø–∏–Ω—è—î—Ç—å—Å—è.

3.  **–§–∞–∑–∞ –æ—á–∏—â–µ–Ω–Ω—è (Sweep phase):** GC –ø—Ä–æ—Ö–æ–¥–∏—Ç—å –ø–æ –≤—Å—ñ–π –∫—É–ø—ñ (heap) —ñ –≤–∏–¥–∞–ª—è—î –≤—Å—ñ –æ–±'—î–∫—Ç–∏, —è–∫—ñ –Ω–µ –±—É–ª–∏ –ø–æ–∑–Ω–∞—á–µ–Ω—ñ. 
4.  –ó–Ω—ñ–º–∞—é—Ç—å—Å—è –≤—Å—ñ –ø–æ–∑–Ω–∞—á–∫–∏, —ñ —Ä–æ–±–æ—Ç–∞ –ø—Ä–æ–≥—Ä–∞–º–∏ –≤—ñ–¥–Ω–æ–≤–ª—é—î—Ç—å—Å—è. 

**–ü—Ä–æ–±–ª–µ–º–∏**
* **"Stop the world":** –ü—Ä–∏–∑—É–ø–∏–Ω–∫–∞ –ø—Ä–æ–≥—Ä–∞–º–∏ –º–æ–∂–µ –≤–∏–∫–ª–∏–∫–∞—Ç–∏ –ø–æ–º—ñ—Ç–Ω—ñ –∑–∞—Ç—Ä–∏–º–∫–∏ ("–ª–∞–≥–∏"), –æ—Å–æ–±–ª–∏–≤–æ –≤ —ñ–Ω—Ç–µ—Ä–∞–∫—Ç–∏–≤–Ω–∏—Ö –¥–æ–¥–∞—Ç–∫–∞—Ö. 
* **–ü–æ–≤—ñ–ª—å–Ω–∞ —Ä–æ–±–æ—Ç–∞ –Ω–∞ –≤–µ–ª–∏–∫–∏—Ö –∫—É–ø–∞—Ö:** –¶—é –ø—Ä–æ–±–ª–µ–º—É –≤–∏—Ä—ñ—à—É—é—Ç—å **–≥–µ–Ω–µ—Ä–∞—Ü—ñ–π–Ω—ñ –∑–±–∏—Ä–∞—á—ñ —Å–º—ñ—Ç—Ç—è (generational collectors)**. 
* **–§—Ä–∞–≥–º–µ–Ω—Ç–∞—Ü—ñ—è –ø–∞–º'—è—Ç—ñ:** –ü—ñ—Å–ª—è –≤–∏–¥–∞–ª–µ–Ω–Ω—è –æ–±'—î–∫—Ç—ñ–≤ —É –∫—É–ø—ñ –º–æ–∂—É—Ç—å –∑–∞–ª–∏—à–∞—Ç–∏—Å—è "–¥—ñ—Ä–∫–∏", —â–æ —É—Å–∫–ª–∞–¥–Ω—é—î –≤–∏–¥—ñ–ª–µ–Ω–Ω—è –ø–∞–º'—è—Ç—ñ –¥–ª—è –≤–µ–ª–∏–∫–∏—Ö –Ω–æ–≤–∏—Ö –æ–±'—î–∫—Ç—ñ–≤. –¶—é –ø—Ä–æ–±–ª–µ–º—É –≤–∏—Ä—ñ—à—É—é—Ç—å **—É—â—ñ–ª—å–Ω—é—é—á—ñ –∑–±–∏—Ä–∞—á—ñ —Å–º—ñ—Ç—Ç—è (compacting collectors)**.

### –†—É—à—ñ–π –≤–∏–∫–æ–Ω–∞–Ω–Ω—è: –Ü–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä —Ç–∞ –∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–∏ VM

**–Ü–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä** ‚Äî —Ü–µ —á–∞—Å—Ç–∏–Ω–∞ —Ä—É—à—ñ—è –≤–∏–∫–æ–Ω–∞–Ω–Ω—è (Execution Engine), —è–∫–∞ –ø—Ä–∞—Ü—é—î —è–∫ –≤—ñ—Ä—Ç—É–∞–ª—å–Ω–∏–π –ø—Ä–æ—Ü–µ—Å–æ—Ä. –ô–æ–≥–æ –∑–∞–≤–¥–∞–Ω–Ω—è ‚Äî –¥–µ–∫–æ–¥—É–≤–∞—Ç–∏ —Ç–∞ –≤–∏–∫–æ–Ω—É–≤–∞—Ç–∏ –≤—ñ—Ä—Ç—É–∞–ª—å–Ω—ñ, –∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–Ω–æ-–Ω–µ–π—Ç—Ä–∞–ª—å–Ω—ñ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó (–±–∞–π—Ç-–∫–æ–¥) –±–µ–∑ –ø–æ–ø–µ—Ä–µ–¥–Ω—å–æ—ó –∫–æ–º–ø—ñ–ª—è—Ü—ñ—ó –≤ –Ω–∞—Ç–∏–≤–Ω–∏–π –º–∞—à–∏–Ω–Ω–∏–π –∫–æ–¥. –ö–æ–¥, —â–æ –≤–∏–∫–æ–Ω—É—î—Ç—å—Å—è —ñ–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä–æ–º, –ø—Ä–∞—Ü—é—î –ø–æ–≤—ñ–ª—å–Ω—ñ—à–µ, –Ω—ñ–∂ —Å–∫–æ–º–ø—ñ–ª—å–æ–≤–∞–Ω–∏–π –Ω–∞—Ç–∏–≤–Ω–∏–π –∫–æ–¥. 

**–û—Å–Ω–æ–≤–Ω—ñ –∫–æ–Ω—Ü–µ–ø—Ü—ñ—ó —ñ–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä–∞:** 
* –í—ñ–Ω –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î —Å—Ç—Ä—É–∫—Ç—É—Ä–∏ –¥–∞–Ω–∏—Ö –¥–ª—è –∑–±–µ—Ä—ñ–≥–∞–Ω–Ω—è —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ–π —Ç–∞ –æ–ø–µ—Ä–∞–Ω–¥—ñ–≤ (–¥–∞–Ω–∏—Ö, —â–æ –æ–±—Ä–æ–±–ª—è—é—Ç—å—Å—è). 
* –î–ª—è –æ–ø–µ—Ä–∞—Ü—ñ–π –≤–∏–∫–ª–∏–∫—É —Ñ—É–Ω–∫—Ü—ñ–π –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—î—Ç—å—Å—è **—Å—Ç–µ–∫ –≤–∏–∫–ª–∏–∫—ñ–≤ (call stack)**. 
* **–õ—ñ—á–∏–ª—å–Ω–∏–∫ –∫–æ–º–∞–Ω–¥ (Instruction Pointer)** –≤–∫–∞–∑—É—î –Ω–∞ –Ω–∞—Å—Ç—É–ø–Ω—É —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—é, —è–∫—É –ø–æ—Ç—Ä—ñ–±–Ω–æ –≤–∏–∫–æ–Ω–∞—Ç–∏. 
* **–í—ñ—Ä—Ç—É–∞–ª—å–Ω–∏–π "CPU" (–¥–∏—Å–ø–µ—Ç—á–µ—Ä —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ–π)** –≤—ñ–¥–ø–æ–≤—ñ–¥–∞—î –∑–∞ –æ—Ç—Ä–∏–º–∞–Ω–Ω—è, –¥–µ–∫–æ–¥—É–≤–∞–Ω–Ω—è —Ç–∞ –≤–∏–∫–æ–Ω–∞–Ω–Ω—è –∫–æ–∂–Ω–æ—ó —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó. 

–ê—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–∞ –≤—ñ—Ä—Ç—É–∞–ª—å–Ω–æ—ó –º–∞—à–∏–Ω–∏ –≤–∏–∑–Ω–∞—á–∞—î, —è–∫ –≤–æ–Ω–∞ –ø—Ä–∞—Ü—é—î –∑ –æ–ø–µ—Ä–∞–Ω–¥–∞–º–∏ –¥–ª—è —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ–π. –Ü—Å–Ω—É—é—Ç—å –¥–≤–∞ –æ—Å–Ω–æ–≤–Ω—ñ –ø—ñ–¥—Ö–æ–¥–∏: —Å—Ç–µ–∫–æ–≤–∏–π —Ç–∞ —Ä–µ–≥—ñ—Å—Ç—Ä–æ–≤–∏–π.

### –°—Ç–µ–∫–æ–≤–∞ –∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–∞ VM (Stack-based)

–¶–µ –∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–∞, —è–∫—É –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å Java, Smalltalk —Ç–∞ –±—ñ–ª—å—à—ñ—Å—Ç—å —ñ–Ω—à–∏—Ö –≤—ñ—Ä—Ç—É–∞–ª—å–Ω–∏—Ö –º–∞—à–∏–Ω. 
* **–ü—Ä–∏–Ω—Ü–∏–ø —Ä–æ–±–æ—Ç–∏:** –í—Å—ñ –æ–ø–µ—Ä–∞–Ω–¥–∏ –¥–ª—è –æ–ø–µ—Ä–∞—Ü—ñ–π –∑–±–µ—Ä—ñ–≥–∞—é—Ç—å—Å—è —É —Å—Ç–µ–∫—É. –Ü–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó –±–µ—Ä—É—Ç—å –∑–Ω–∞—á–µ–Ω–Ω—è –∑ –≤–µ—Ä—à–∏–Ω–∏ —Å—Ç–µ–∫—É, –æ–±—Ä–æ–±–ª—è—é—Ç—å —ó—Ö —ñ –∫–ª–∞–¥—É—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç –Ω–∞–∑–∞–¥ –Ω–∞ –≤–µ—Ä—à–∏–Ω—É —Å—Ç–µ–∫—É –∑–∞ –ø—Ä–∏–Ω—Ü–∏–ø–æ–º **LIFO (Last in, First Out)**.

#### –ü–µ—Ä–µ–≤–∞–≥–∏ —Ç–∞ –Ω–µ–¥–æ–ª—ñ–∫–∏
* **–ü–µ—Ä–µ–≤–∞–≥–∏:**
    * –ü—Ä–æ—Å—Ç–æ—Ç–∞ —Ä–µ–∞–ª—ñ–∑–∞—Ü—ñ—ó –¥–ª—è —Ä—ñ–∑–Ω–∏—Ö –∞–ø–∞—Ä–∞—Ç–Ω–∏—Ö –∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä. 
    * –õ–µ–≥–∫–æ –∞–¥—Ä–µ—Å—É–≤–∞—Ç–∏ –æ–ø–µ—Ä–∞–Ω–¥–∏ –∑–∞ –¥–æ–ø–æ–º–æ–≥–æ—é –≤–∫–∞–∑—ñ–≤–Ω–∏–∫–∞ —Å—Ç–µ–∫—É; VM –Ω–µ –ø–æ—Ç—Ä—ñ–±–Ω–æ –∑–Ω–∞—Ç–∏ —ó—Ö–Ω—ñ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ñ –∞–¥—Ä–µ—Å–∏. 
* **–ù–µ–¥–æ–ª—ñ–∫–∏:**
    * –î–ª—è –≤–∏–∫–æ–Ω–∞–Ω–Ω—è –æ–¥–Ω—ñ—î—ó –ø—Ä–æ—Å—Ç–æ—ó –æ–ø–µ—Ä–∞—Ü—ñ—ó (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –¥–æ–¥–∞–≤–∞–Ω–Ω—è) –ø–æ—Ç—Ä—ñ–±–Ω–æ –≤–∏–∫–æ–Ω–∞—Ç–∏ –∫—ñ–ª—å–∫–∞ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ–π (–∑–∞–≤–∞–Ω—Ç–∞–∂–∏—Ç–∏ –ø–µ—Ä—à–∏–π –æ–ø–µ—Ä–∞–Ω–¥, –∑–∞–≤–∞–Ω—Ç–∞–∂–∏—Ç–∏ –¥—Ä—É–≥–∏–π, –≤–∏–∫–æ–Ω–∞—Ç–∏ –æ–ø–µ—Ä–∞—Ü—ñ—é, –∑–±–µ—Ä–µ–≥—Ç–∏ —Ä–µ–∑—É–ª—å—Ç–∞—Ç), —â–æ —î –º–µ–Ω—à –µ—Ñ–µ–∫—Ç–∏–≤–Ω–∏–º.

### –†–µ–≥—ñ—Å—Ç—Ä–æ–≤–∞ –∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä–∞ VM (Register-based)

–¶–µ–π –ø—ñ–¥—Ö—ñ–¥ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å –Ω–µ–±–∞–≥–∞—Ç–æ VM, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥, Lua, Dalvik (—Å—Ç–∞—Ä–∞ VM –≤ Android) —Ç–∞ Parrot VM (Perl6). 
* **–ü—Ä–∏–Ω—Ü–∏–ø —Ä–æ–±–æ—Ç–∏:** –û–ø–µ—Ä–∞–Ω–¥–∏ –∑–±–µ—Ä—ñ–≥–∞—é—Ç—å—Å—è —É –≤—ñ—Ä—Ç—É–∞–ª—å–Ω–∏—Ö —Ä–µ–≥—ñ—Å—Ç—Ä–∞—Ö, –ø–æ–¥—ñ–±–Ω–æ –¥–æ —Ç–æ–≥–æ, —è–∫ —Ü–µ –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è —É —Ä–µ–∞–ª—å–Ω–æ–º—É CPU. –ö–æ–∂–Ω–∞ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—è –º—ñ—Å—Ç–∏—Ç—å –∞–¥—Ä–µ—Å–∏ (–Ω–æ–º–µ—Ä–∏ —Ä–µ–≥—ñ—Å—Ç—Ä—ñ–≤) –æ–ø–µ—Ä–∞–Ω–¥—ñ–≤, –∑ —è–∫–∏–º–∏ –≤–æ–Ω–∞ –ø—Ä–∞—Ü—é—î.

#### –ü–µ—Ä–µ–≤–∞–≥–∏ —Ç–∞ –Ω–µ–¥–æ–ª—ñ–∫–∏
* **–ü–µ—Ä–µ–≤–∞–≥–∏:**
    * –ü–æ—Ç–µ–Ω—Ü—ñ–π–Ω–æ —à–≤–∏–¥—à–∞ –∑–∞ —Å—Ç–µ–∫–æ–≤—É –∞—Ä—Ö—ñ—Ç–µ–∫—Ç—É—Ä—É. 
    * –ù–µ–º–∞—î –Ω–∞–∫–ª–∞–¥–Ω–∏—Ö –≤–∏—Ç—Ä–∞—Ç –Ω–∞ –æ–ø–µ—Ä–∞—Ü—ñ—ó push/pop –∑—ñ —Å—Ç–µ–∫—É, —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó –≤–∏–∫–æ–Ω—É—é—Ç—å—Å—è —à–≤–∏–¥—à–µ. 
    * –ü–æ—Ç—Ä—ñ–±–Ω–æ –º–µ–Ω—à–µ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ–π –¥–ª—è –æ–¥–Ω—ñ—î—ó –æ–ø–µ—Ä–∞—Ü—ñ—ó. 
    * –õ–µ–≥—à–µ –æ–ø—Ç–∏–º—ñ–∑—É–≤–∞—Ç–∏ –ø—ñ–¥–ø—Ä–æ–≥—Ä–∞–º–∏, –æ—Å–∫—ñ–ª—å–∫–∏ —Ç–∏–º—á–∞—Å–æ–≤—ñ –∑–Ω–∞—á–µ–Ω–Ω—è –∑–∞–∑–≤–∏—á–∞–π –∑–∞–ª–∏—à–∞—é—Ç—å—Å—è –≤ —Ä–µ–≥—ñ—Å—Ç—Ä–∞—Ö. 
* **–ù–µ–¥–æ–ª—ñ–∫–∏:**
    * –ú–µ–Ω—à –ø–æ—Ä—Ç–∞—Ç–∏–≤–Ω–∞, –æ—Å–∫—ñ–ª—å–∫–∏ —Ä–æ–±–∏—Ç—å –±—ñ–ª—å—à–µ –ø—Ä–∏–ø—É—â–µ–Ω—å —â–æ–¥–æ —Ü—ñ–ª—å–æ–≤–æ–≥–æ "–∑–∞–ª—ñ–∑–∞". 
    * –Ü–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó –±—ñ–ª—å—à—ñ –∑–∞ —Ä–æ–∑–º—ñ—Ä–æ–º, –æ—Å–∫—ñ–ª—å–∫–∏ —ó–º –ø–æ—Ç—Ä—ñ–±–Ω–æ —è–≤–Ω–æ –≤–∫–∞–∑—É–≤–∞—Ç–∏ –∞–¥—Ä–µ—Å–∏ –æ–ø–µ—Ä–∞–Ω–¥—ñ–≤.

### –û–ø—Ç–∏–º—ñ–∑–∞—Ü—ñ—è –≤–∏–∫–æ–Ω–∞–Ω–Ω—è: JIT —Ç–∞ AOT –∫–æ–º–ø—ñ–ª—è—Ü—ñ—è

–Ü–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü—ñ—è –±–∞–π—Ç-–∫–æ–¥—É —î –≥–Ω—É—á–∫–æ—é, –∞–ª–µ –ø–æ–≤—ñ–ª—å–Ω–æ—é. –î–ª—è –≤–∏—Ä—ñ—à–µ–Ω–Ω—è —Ü—ñ—î—ó –ø—Ä–æ–±–ª–µ–º–∏ —ñ—Å–Ω—É—é—Ç—å –±—ñ–ª—å—à –ø—Ä–æ—Å—É–Ω—É—Ç—ñ —Ç–µ—Ö–Ω—ñ–∫–∏.

#### JIT (Just-In-Time) –∫–æ–º–ø—ñ–ª—è—Ü—ñ—è
* –¶–µ –º–µ—Ö–∞–Ω—ñ–∑–º –¥–ª—è –ø–æ–∫—Ä–∞—â–µ–Ω–Ω—è –ø—Ä–æ–¥—É–∫—Ç–∏–≤–Ω–æ—Å—Ç—ñ –ø—Ä–æ–≥—Ä–∞–º, —â–æ –ø—Ä–∞—Ü—é—é—Ç—å –Ω–∞ –æ—Å–Ω–æ–≤—ñ –±–∞–π—Ç-–∫–æ–¥—É.
* –ô–æ–≥–æ —Å—É—Ç—å –ø–æ–ª—è–≥–∞—î –≤ —Ç–æ–º—É, —â–æ –≤—ñ—Ä—Ç—É–∞–ª—å–Ω—ñ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó –∫–æ–º–ø—ñ–ª—é—é—Ç—å—Å—è –≤ –Ω–∞—Ç–∏–≤–Ω—ñ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó "–Ω–∞ –ª—å–æ—Ç—É", —Ç–æ–±—Ç–æ –±–µ–∑–ø–æ—Å–µ—Ä–µ–¥–Ω—å–æ –ø–µ—Ä–µ–¥ –∞–±–æ –ø—ñ–¥ —á–∞—Å –≤–∏–∫–æ–Ω–∞–Ω–Ω—è –ø—Ä–æ–≥—Ä–∞–º–∏.
* JIT-–∫–æ–º–ø—ñ–ª—è—Ü—ñ—è –∑–∞–∑–≤–∏—á–∞–π –≤–∏–∫–ª–∏–∫–∞—î –Ω–µ–≤–µ–ª–∏–∫—É –∑–∞—Ç—Ä–∏–º–∫—É –Ω–∞ —Å—Ç–∞—Ä—Ç—ñ –ø—Ä–æ–≥—Ä–∞–º–∏ —á–µ—Ä–µ–∑ —á–∞—Å, –Ω–µ–æ–±—Ö—ñ–¥–Ω–∏–π –¥–ª—è –∑–∞–≤–∞–Ω—Ç–∞–∂–µ–Ω–Ω—è —Ç–∞ –∫–æ–º–ø—ñ–ª—è—Ü—ñ—ó –±–∞–π—Ç-–∫–æ–¥—É.

#### AOT (Ahead-Of-Time) –∫–æ–º–ø—ñ–ª—è—Ü—ñ—è
* –ü—Ä–∏ —Ü—å–æ–º—É –ø—ñ–¥—Ö–æ–¥—ñ –±–∞–π—Ç-–∫–æ–¥ –∫–æ–º–ø—ñ–ª—é—î—Ç—å—Å—è –≤ –Ω–∞—Ç–∏–≤–Ω—ñ —ñ–Ω—Å—Ç—Ä—É–∫—Ü—ñ—ó –∑–∞–∑–¥–∞–ª–µ–≥—ñ–¥—å, **–¥–æ** –∑–∞–ø—É—Å–∫—É –ø—Ä–æ–≥—Ä–∞–º–∏.
* –ü—Ä–æ–≥—Ä–∞–º–∞, —â–æ —Å–∫–æ–º–ø—ñ–ª—å–æ–≤–∞–Ω–∞ –∑–∞–∑–¥–∞–ª–µ–≥—ñ–¥—å (AOT), –º–æ–∂–µ –∑–∞–ø—É—Å–∫–∞—Ç–∏—Å—è —à–≤–∏–¥—à–µ, –Ω—ñ–∂ —Ç–∞, —â–æ –ø–æ–∫–ª–∞–¥–∞—î—Ç—å—Å—è –≤–∏–∫–ª—é—á–Ω–æ –Ω–∞ JIT-–∫–æ–º–ø—ñ–ª—è—Ü—ñ—é.
